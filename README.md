# homebrew-podspine

Homebrew tap for [Podspine](https://github.com/schubydoo/podspine) — a self-hosted
server that turns audiobooks into per-chapter podcast feeds.

```bash
brew install schubydoo/podspine/podspine
# or:
brew tap schubydoo/podspine
brew install podspine
```

`Formula/podspine.rb` is **auto-synced** from the canonical formula in the main repo
(`schubydoo/podspine`) by `.github/workflows/sync-formula.yml` — don't hand-edit it here.
